# База знаний — «не работает интернет»

25 SOP-документов для оператора телеком-техподдержки.

**Источник**: документы составлены вручную на основе публичных FAQ российских операторов (МТС, Билайн, МегаФон, Tele2) и типовых сценариев колл-центра. **Это не дословные тексты с сайтов операторов**, а синтезированные регламенты, имитирующие реальную KB. Подход согласован с целью курсовой — оценить качество RAG-пайплайна на реалистичной по структуре, но открытой по лицензии базе.

## Структура файла

Каждый документ имеет YAML-frontmatter с метаданными:

```yaml
---
doc_id: <уникальный slug>
title: <заголовок документа>
intents: [<intent1>, <intent2>, ...]
emotion_context: <neutral|angry|frustrated|all>
---
```

И тело из секций:

- **Когда применять** — триггер-фразы клиента и ситуации
- **Действия оператора** — пошаговая диагностика
- **Рекомендованный ответ** — что сказать клиенту дословно
- **Эскалация** — когда передать на L2 или оформить выезд техника

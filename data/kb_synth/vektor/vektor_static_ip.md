# Статический IP-адрес

По умолчанию домашний интернет «Вектор» использует динамический IP. Статический IP нужен для видеонаблюдения, доступа к домашнему серверу, удалённого рабочего стола. Подключается как платная опция.

## Диагностика
Уточните цель: для большинства задач (камеры, NAS) достаточно статического «белого» IP. Проверьте, что у клиента активный тариф без задолженности.

## Шаги решения
1. Оформите подключение опции «Статический IP» в CRM.
2. После активации (до 1 часа) клиенту назначается фиксированный белый IP.
3. На «Вектор Box» статический IP подтянется автоматически через PPPoE — ручная настройка обычно не требуется.
4. Для проброса портов: 192.168.1.1 → «Дополнительно» → «Перенаправление портов».

## Что сказать клиенту
«Статический IP стоит 150 ₽ в месяц. После подключения у вас будет постоянный белый адрес — к камерам или серверу можно будет заходить из любой точки. Настройку проброса портов я подскажу.»

## Условия/тарификация
- Статический IP — **150 ₽/мес**, активация бесплатно.
- Опция доступна на всех тарифах домашнего интернета.

## Если не помогло
Если после активации внешний доступ не работает — проверьте проброс портов и брандмауэр устройства. Уточните, не использует ли клиент сторонний роутер за «Вектор Box» (двойной NAT).

## Эскалация
Проблемы с маршрутизацией белого IP — заявка в технический отдел L2.
